# Chronic Pain Data

A reduced version of the data collected by Hechler et al. (2014)

## Usage

``` r
hechler_2014
```

## Format

A tibble with 208 rows and 3 variables:

- `patient`:

  Patient identifier

- `measurement`:

  Indicator of measurement

- `disability`:

  Pain-related disability as measured with the PPDI (lower is better)

## References

Hechler, T., Ruhe, A.‑K., Schmidt, P., Hirsch, J., Wager, J., Dobe, M.,
Krummenauer, F., & Zernikow, B. (2014). Inpatient Based Intensive
Interdisciplinary Pain Treatment for Highly Impaired Children with
Severe Chronic Pain: Randomized Controlled Trial of Efficacy and
Economic Effects. Pain, 155(1), 118–128.
https://doi.org/10.1016/j.pain.2013.09.015
